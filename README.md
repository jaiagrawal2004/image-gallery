# 🖼️ Image Gallery

A responsive and modern Image Gallery website built using HTML, CSS, and JavaScript. The project allows users to view beautiful images in a clean grid layout with an attractive user interface.

## 🚀 Features

* Responsive image gallery layout
* Modern and clean UI design
* Hover effects and animations
* Mobile-friendly design
* Fast and lightweight
* Easy to customize

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```bash
Image-Gallery/
│── index.html
│── style.css
│── script.js
└── images/
```


## 🔧 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/image-gallery.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
