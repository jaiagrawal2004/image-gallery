Project Description

This Responsive Image Gallery is developed using HTML, CSS, and JavaScript. It allows users to browse images using category filters, search functionality, dark/light mode, and a lightbox viewer with next and previous navigation. The gallery uses a modern glassmorphism design and masonry layout to provide an attractive and responsive user experience on all devices.
